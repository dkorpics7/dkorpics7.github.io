Hawaii In Review

Description:

This project is a travel review site for the Hawaiian Islands. It was set up as a code-first MVC application with a database to store user reviews. A second table in the database was used to store the islands being reviewed. The purpose of the application was to allow users to review an attraction or event, see previous reviews, and update or delete these reviews. They could also enter, update, and delete the island names, with the exception of the main eight Hawaiian islands which I did not allow the user to delete. Please scroll down to see various features of the application.

Skills Used:

ASP.NET, MVC (code-first database), HTML5, CSS3, BOOTSTRAP

The Home page tells about the application and contains general information about the main islands.

From the "Write a Review" page, the user can enter a new review.

Once reviews have been entered, they can be viewed.

If you click on the button to "View Details", you can see the entire review.

From the detailed review, you may update the review...

... or delete it.

I also added a feature to list all the detailed reviews at once.

The user also has the ability to display the current reviewed islands.

From here, they can update or delete the island.

And of course, there is a contact page.
