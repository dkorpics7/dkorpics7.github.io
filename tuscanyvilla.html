Tuscany Villa

Description:

This project was my first web site! The goal was to use my newfound HTML and CSS skills to copy a design made by my instructor. From this exercise, I learned how to position images and text (without bootstrap!), format pages, create forms, and make a carousel. Please scroll down to see the various features.

Skills Used:

ASP.NET, HTML5, CSS3

The Home page is two pictures side by side with text overlaid on top.

The entire web site is a single HTML file. Therefore, you can either scroll down or click on the navigation menu to get to "Locations". This page was three rows with alternating background colors and thumbnail pictures.

Scrolling down again, we come to the "Amenities" page. This page is laid out in a tic-tac-toe pattern with alternating images and text.

The next page is a review page with a carousel with various customer reviews.

Finally, the last page contains a contact form and a footer.
