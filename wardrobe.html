What's In YOUR Closet?

Description:

This project was designed to help you organize the items in your closet. It was set up as a database-first MVC application with MS SQL. Eight tables were designed. The first four tables were to store tops, bottoms, shoes, and accessories. The fifth table was designed with a foreign key from each of those categories to allow the user to design outfits from their wardrobe. The last three tables were used to help the user enter the color, season, and occasion for each article of clothing. Please scroll down to see the various features.

Skills Used:

ASP.NET, MVC, MS SQL, HTML5, CSS3, BOOTSTRAP
